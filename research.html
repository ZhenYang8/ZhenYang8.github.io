Unmanned Systems — Perception

2D LiDAR SLAM

We developed a multi-robot cooperative exploration and mapping system based on 2D LiDAR. The system employs a greedy algorithm combined with RRT frontier exploration for fine-grained search and target point assignment, and integrates a map fusion algorithm for real-time local map merging.

3D LiDAR SLAM

Our 3D LiDAR SLAM system incorporates global descriptor-based place recognition and iterative closest point (ICP) matching for backend optimization, effectively eliminating accumulated errors and enhancing localization and mapping accuracy. For multi-robot scenarios, the system adopts a centralized architecture using KD-tree for map data association and inter-map transformation estimation, enabling efficient point cloud map construction in large-scale environments such as underground garages and outdoor campuses.

Dynamic Semantic Visual SLAM

Our visual SLAM system integrates lightweight semantic segmentation with geometric constraints to achieve robust localization in dynamic environments, while reconstructing dense static point cloud maps with improved accuracy and real-time performance.

Unmanned Systems—Motion Planning

Formation Control

Behavior-Based Formation Control

The robot motion is decomposed into three behaviors: formation keeping, goal seeking, and collision avoidance, enabling robots with nonholonomic constraints to traverse obstacle regions while maintaining formation and reach the target point, and to autonomously switch to appropriate formations based on the current environment.

   
Gazebo simulation: multi-robot formation control Real-world experiment (Mecanum wheel robot platform)

Multi-Robot Formation Control Based on Artificial Potential Field and Fuzzy Control

Our formation control system combines artificial potential field (APF) method with fuzzy control to achieve obstacle avoidance capability and optimized path planning, effectively circumventing the local minimum issues inherent in traditional APF algorithms. The approach has been validated through real-world experiments on Mecanum wheel robot platforms.

Autonomous Decision-Making and Planning

End-to-End Navigation (Static Unknown Scenes)

Our navigation system supports both single-agent and multi-agent scenarios in static unknown environments. The single-agent module employs radar-based map-free reinforcement learning, while the multi-agent module integrates artificial potential field with reinforcement learning for coordinated path planning, enabling efficient policy training and execution.

Multi-Robot Path Planning

Conflict-Based Search

Our multi-robot path planning framework adopts a hierarchical conflict-based search (CBS) algorithm, progressively introducing path constraints to find conflict-free solutions. The framework is scalable to heterogeneous multi-robot systems with varying models and body sizes.

Simulation and Testing Platform

Electricity-Hydrogen Integrated Energy System Simulation and Testing Platform

We independently developed a large-scale simulation platform for electricity-hydrogen integrated energy systems based on B/S architecture, integrating system-level modeling of wind power, photovoltaics, lithium battery storage, ALK/PEM electrolyzers, fuel cells, and distribution networks. The platform enables device-level parameter configuration and supports real-time dynamic simulation at second-level scale using advanced solvers such as Ipopt. It also incorporates multi-time-scale scheduling strategies (day-ahead, intra-day, and steady-state optimization) to optimize power allocation across heterogeneous electrolyzer clusters and multi-type storage systems.
